# 控制

## 介绍

本模块基于规划和当前的汽车状态，使用不同的控制算法来生成舒适的驾驶体验。控制模块可以在正常模式和导航模式下工作。

## 输入

* 规划轨迹
* 车辆状态
* 定位
* Dreamview自动模式更改请求
## 输出

* 给底盘的控制指令（转向，节流，刹车）。
